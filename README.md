# zerotier私有根一键替换脚本

## windows系统

- 请使用管理员打开cmd，执行脚本

```cmd
# 国外
curl -L -o update-planet.cmd https://raw.githubusercontent.com/dadastory/zerotier-replace-custom/main/update-planet.cmd && update-planet.cmd https://your.domain/planet.custom

# 国内镜像
curl -L -o update-planet.cmd https://gh.llkk.cc/https://raw.githubusercontent.com/dadastory/zerotier-replace-custom/main/update-planet.cmd && update-planet.cmd https://your.domain/planet.custom
```
